# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), with one Vietnam-specific addition.

## TL;DR

- Be respectful. Disagreement is fine; contempt is not.
- Welcome contributors regardless of background, nationality, or experience level.
- Cultural sensitivity matters here more than most projects — see below.

## Vietnamese cultural context — additional norms

This is a project ABOUT Vietnam, BUILT WITH Vietnamese contributors and welcoming foreign contributors. The following are not negotiable in our discussions, issues, PRs, or documentation:

1. **"Sài Gòn" vs "TP.HCM"** — both terms are used in this project (persona voice uses Sài Gòn, admin labels use TP.HCM). Do not use either term to make political points or to disparage the other usage.
2. **Royal history (Huế)** — present with respect; no Disneyfication or mockery.
3. **War heritage (Hải Phòng 1972, others)** — factual only; do not glorify any side.
4. **Post-2025 administrative reform** — neutral facts only. Some users miss the old province names; some don't. Do not editorialize about the merger.
5. **Ethnic communities** — name accurately (K'Ho, M'Nông, Churu, Chăm, Khmer Krom, Kinh, etc.). Don't generalize.
6. **Diaspora vs domestic identity** — both are legitimate Vietnamese experiences. Do not gatekeep "Vietnamese-ness" on either side.

If a PR or issue violates these, a maintainer may close it. We will explain why and welcome the contributor to refile after revising — this is a teaching moment, not a strike.

## Enforcement

Report Code of Conduct concerns to `dung@newera.inc`. Reports are confidential.

Enforcement ladder (Contributor Covenant 2.1 default):
1. Correction (private note)
2. Warning (public if pattern repeats)
3. Temporary ban
4. Permanent ban

Maintainers act in good faith and explain decisions when feasible.
